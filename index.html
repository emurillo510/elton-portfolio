ABOUT

JavaScript is one of my favorite languages to write. Coming from a Java background, I find that working on JavaScript projects is more liberating and expressive than Java. Lately, I have been learning many modern frameworks and libraries for JavaScript such as Ember.JS and Backbone.JS + Marionette; two extremely great technologies for building applications.

Ruby on Rails is my first general-purpose language outside of Java. I started learning it due to starting Coppis on Ruby on Rails. What makes Ruby great is that it is truly more object-oriented than Java and it also has a more human like syntax due to how it handles loose typing. I try to use Ruby now more than Java as my go to language. I am also aim to learn Python.

PORTFOLIO

Coppis was my first web application using a modern framework; Ruby on Rails. Coppis is a fashion platform where entrepreneurs can post trending urban fashion. This platform allows the community to see up and trending clothing from local businesses. Think instagram for local urban clothing. This project allowed me to learn many various technologies and techniques such as MVC. We touched many RubyGems such as Devise, Paperclip, Rspec, and minitest just to name a few. Also, it forced us to learn a new language (Ruby). Ever since Ruby has become one of my commonly used languages on a day-to-day basis. We learned about Object-Relational Mapping and also learned how to configure and deploy servers using Nginx and Postgres. Even though many people are transitioning to a Node.js backend; Ruby has a great community and both Ruby and Rails are continually being optimized. I am proud in using Ruby on Rails and I am proud to have been apart of Coppis.

I am currently writing my own Bart application for my daily commute. It will tell me when the next trains are coming. The application pulls from the part API which structures its data in XML. A Rails backend with a Bart API key pulls the data and sends it across the network via JSON. A Backbone Frontend picks up the data and renders its content. User data is stored in the browsers localStorage interface so data is persisted. This application allows me to get to work on time!
